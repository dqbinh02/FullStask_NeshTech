1. Git status
            Trạng thái hiện tại respo đang đứng
        2. git branch 
            Tạo một branch mới từ nhánh hiện hành
        3. git checkout 
            Chuyển trạng thái một nhánh đã tạo
        4. git commit -m ""
            Tạo một cam kết khi chúng ta xác định một file hoặc nhiều file thay đổi
        5. git push origin 
            Đẩy những thay đổi mới nhất mà chúng ta đã cam kết lên origin
        6. git add
